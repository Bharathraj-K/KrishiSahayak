# KrishiSahayak Assets

This directory contains all the assets for the KrishiSahayak app.

## Required Assets

To complete the app setup, please add the following assets:

### Icons and Images
- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `splash.png` - Splash screen image (1284x2778px)
- `favicon.png` - Web favicon (16x16px or 32x32px)

### Color Scheme
The app uses an agricultural green theme:
- Primary: #2E7D32 (Green)
- Secondary: #4CAF50 (Light Green)
- Accent: #8BC34A (Lime Green)
- Background: #F1F8E9 (Very Light Green)

### Asset Guidelines
- Use high-quality images suitable for different screen densities
- Maintain consistent branding across all assets
- Follow platform-specific guidelines for icon design
- Optimize images for mobile performance

## Asset Sources
You can create or source assets from:
- Custom design tools (Figma, Sketch, etc.)
- Icon libraries (Material Icons, Feather Icons)
- Agricultural imagery from free stock photo sites
- Custom illustrations focusing on Indian agriculture

## Note
The app will work without these assets, but proper assets will enhance the user experience and make the app production-ready.
